# Colorful Datestamp API

A public OpenAPI 3.1.0 specification for Brunei Darussalam's ICT compliance metadata,
datestamp formatting tokens, and ISO standards — for use in ChatGPT Custom GPTs.

## Usage

Deploy this to GitHub Pages and paste the raw `.yaml` link into ChatGPT’s “Actions” tab.

## Endpoints

- `/formats` – Date format patterns
- `/token` – Global token for UI
- `/regulator` – Regulator info (AITI)
- `/compliance/mobile` – ISO standards for apps
